Fish Pin

I live in a lake community, so naturally I want a way to keep track of my success on the lake. That’s where the idea for Fish Pin came in. Fish Pin allows the user the ability to add details about each catch including the fish type, size, lure, weather, time, and notes. Each entry is saved with latitude and longitude and displayed with a custom annotation on the map using MapKit. In order to make data entry easier, I decided to use a weather API to automatically enter in the temperature and weather description (i.e. sunny, cloudy, etc.). The date and time is also entered automatically.

Fish Pin is still a work in progress, but the end result will store all the data using Core Data and display the records in a UICollection View along with a picture taken at the time of the record entry. The map screen displays all the annotations from catches so it’s easy to see where the honey holes are!

For now you can check it out here on GitHub

Features:

  • A Navigation Controller handles the navigation flow of the app
  • Firebase authentication creates user profiles and allows login to the app
  • MapKit is used for showing pins of the user's location and all the recorded catches
  • UITextFields allow the user to enter the data for each catch
  • A weather API pulls in weather based on the user location and auto fills the text fields
  • Date and time fields are automatically filled base on the user's current timezone
  • Core Data is used for data persistence
  • A collection view shows all of the records with the picture taken at the time of the catch

    • API's, Libraries, and Frameworks used in this project:

  • MapKit
  • Firebase
